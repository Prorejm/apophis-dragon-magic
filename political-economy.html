第一章 商品和货币

"资本主义生产方式统治下社会的财富，表现为'一个惊人庞大的商品堆积'，一个一个的商品表现为它的原素形式。所以，我们的研究要从商品的分析开始。" —— 马克思《资本论》第一卷

商品的二因素：使用价值和价值

使用价值

物品的有用性，即它能满足人们某种需要的这种属性，便是物品的使用价值。使用价值是商品的自然属性，这种属性不仅商品有，商品以外的许多物品也是有的。

一件物品要成为商品，首先必须有用，即具有使用价值。但并非一切具有使用价值的东西都是商品。农民供自己消费的粮食有使用价值，但不是商品。一种物品要成为商品，它的使用价值必须是通过买卖形式的交换而进入消费领域。

关键要点：使用价值形成财富的物质内容，无论财富的社会形式怎样，使用价值都形成财富的物质内容。政治经济学是从社会的观点来看待使用价值的。

交换价值与价值

交换价值首先表现为一种使用价值和另一种使用价值相交换的量的关系或量的比例。例如，20斤大米和10尺布相交换，10尺布便是20斤大米的交换价值。

不同商品所以能按一定比例互相交换，是因为在它们里面存在着某种共同的东西——这种共同的东西不能是自然属性，而是人类劳动的凝结，即价值。

核心结论：商品有两种属性（或两个因素）——使用价值和价值。商品是使用价值和价值的统一体。使用价值是商品的自然属性，价值则是商品的社会属性，是商品所特有的属性。

使用价值和价值的矛盾

商品的使用价值和价值处在既相矛盾又相统一的关系中：

  • 统一：使用价值是价值的物质承担者，二者互相依赖、互为条件
  • 矛盾：商品的使用价值是为社会、为别人的，而生产者关心的是价值。商品一经生产出来，生产者必须让渡使用价值才能取得价值

由于商品内部包含着这种矛盾，所以一切商品都必须参加交换过程。只有通过交换，商品的使用价值和价值才能够实现。

劳动的二重性：具体劳动和抽象劳动

生产商品的劳动具有二重性——这是马克思首先发现的，他本人把这一学说看作"在政治经济学的理解上是一个枢纽点"。

具体劳动

各种不同形式的劳动，是根据劳动的目的、对象、所用的工具、操作方法和劳动的结果来区分的。每一种具体形式的劳动，能够生产出一种特定品种的产品，即一个特定的使用价值。

具体劳动作为使用价值的创造者这个根本性质，是不以社会形式的变化为转移的。

抽象劳动

撇开了具体形式的一般的人类劳动，即抽象劳动。商品的价值正是由这种抽象劳动形成的。

"一切劳动，一方面都是人类劳动力生理学意义上的支出。并且，当作等一的人类劳动或抽象的人类劳动，它形成商品价值。一切劳动，另一方面又都是人类劳动力在特殊的有一定目的的形式上的支出。并且，当作具体有用的劳动，它生产使用价值。" —— 马克思

价值量

价值量由社会必要劳动时间决定。所谓社会必要劳动时间，是用社会现有的标准生产条件，用社会平均的劳动熟练程度和强度，生产任何一个使用价值所必要的劳动时间。

商品价值量 = 社会必要劳动时间
劳动生产率与单位商品价值量成反比，与使用价值量成正比

简单劳动与复杂劳动

复杂劳动被看作是强化的或倍加的简单劳动，小量的复杂劳动可以和大量的简单劳动相等。这种比例完全是由一个在商品生产者背后的社会过程自发地确定的。

价值形态的发展与货币的起源

价值形态经历了四个发展阶段，揭示了货币的历史起源：

1. 简单的或偶然的价值形态

1只绵羊 = 2把斧子

绵羊处于相对价值形态，斧子处于等价形态。等价形态有三个特征：使用价值成为价值的现象形态；具体劳动成为抽象劳动的现象形态；私人劳动采取了社会劳动的形态。

2. 扩大的价值形态

1只绵羊 = 2把斧子，或 = 5斤盐，或 = 1克黄金，或 = 3斤茶叶...

商品价值第一次被表现为无差别的人类劳动的凝结，但每一种商品的价值表现都是一个无穷无尽的特殊等价物的序列，还没有共同的统一表现。

3. 一般价值形态

2把斧子 =
5斤盐 =
1克黄金 = 1只绵羊
3斤茶叶 =

一切商品的价值通过一种分离的商品（绵羊）来表现。绵羊成为一般等价物。

4. 货币形态

当一般等价物的机能固定在某种特殊商品（金银）上时，这种商品就成了货币。货币无非是一般等价物，是固定充当一般等价物的商品。

"金银天然不是货币，但货币天然是金银。" —— 马克思

货币的职能

在发达的商品经济条件下，货币执行着五种职能：

1. 价值尺度

货币以自己为尺度来表现和衡量其他一切商品的价值。货币所以能充当价值尺度，是因为货币本身也是商品，具有价值。商品价值表现在货币上，就是商品的价格。货币执行这一职能时是作为观念的货币。

2. 流通手段

商品流通公式：商品—货币—商品（W—G—W）。货币在商品流通中的媒介作用就是流通手段职能。作为流通手段的货币只能是实在的货币。

流通中所需货币量 = 商品价格总额 / 同一单位货币流通的平均速度（次数）

3. 贮藏手段

货币退出流通领域，作为社会财富的一般代表被保存起来，执行贮藏手段的职能。充当贮藏手段的货币必须是足价的金属货币。

4. 支付手段

货币在清偿债务或支付赋税、租金、工资等时的职能。随着信用的发展，货币作为支付手段的职能日益扩大。

5. 世界货币

货币越出国内流通领域，在世界市场上执行一般购买手段、一般支付手段和社会财富的一般化身等职能。

价值规律

价值规律是商品生产的经济规律。它的基本内容是：商品的价值量由生产商品的社会必要劳动时间决定，商品交换以价值为基础，遵循等价交换的原则。

价格围绕价值上下波动，是价值规律作用的表现形式。在私有制商品经济中，价值规律有以下作用：

  • 自发地调节生产资料和劳动力在各部门之间的分配
  • 刺激生产技术的改进和劳动生产率的提高
  • 促使商品生产者两极分化

商品拜物教

商品拜物教是指把商品这种人与人之间的经济关系，看作是物与物之间的关系的虚幻观念。

商品不是物，而是在物的外壳掩盖下的生产者之间的一定的社会关系。人们互相交换商品，实际上是互相交换各自的劳动，只不过由于私人劳动和社会劳动的矛盾，生产者的劳动的社会性不能直接表现出来，因而他们之间的劳动互换才采取了价值的形式、商品的形式。

货币出现后，商品拜物教发展为货币拜物教。在资本主义社会，一切关系都采取了物的形式，拜物教达到了最高峰。

第二章 资本和剩余价值

"资本来到世间，从头到脚，每个毛孔都滴着血和肮脏的东西。" —— 马克思《资本论》第一卷

由货币到资本的转化

资本的总公式

货币是资本的最初表现形式，但货币本身并不是资本。作为一般货币的流通公式是W—G—W（商品—货币—商品），而作为资本的流通公式则是G—W—G'（货币—商品—更多的货币），其中G' = G + ΔG。

G — W — G'   （G' = G + ΔG）
资本总公式

ΔG就是剩余价值。资本就是能够带来剩余价值的价值。G—W—G'这个公式表明，资本运动的目的是为了价值的增殖。

资本总公式的矛盾

剩余价值不能在流通中产生，又不能不在流通中产生。它必须既在流通中又不在流通中产生。这就是资本总公式的矛盾。

解决这一矛盾的关键在于：资本家在市场上找到了一种特殊商品，这种商品的使用价值本身就是价值的源泉——这就是劳动力。

劳动过程和价值增殖过程

资本主义劳动过程

资本主义的劳动过程具有两个特点：

  • 工人在资本家的监督下劳动，他的劳动属于资本家
  • 劳动产品归资本家所有

价值增殖过程

价值增殖过程是超过一定点而延长了的价值形成过程。资本家把工人的劳动时间延长到补偿劳动力价值所需要的劳动时间以上，这个延长部分就是剩余劳动时间，它创造的价值就是剩余价值。

关键区分：必要劳动时间——再生产劳动力价值的时间；剩余劳动时间——创造剩余价值的时间。剩余价值就是由雇佣工人创造的、被资本家无偿占有的、超过劳动力价值以上的那部分价值。

不变资本与可变资本

资本家的资本分为两个部分：

  • 不变资本（c）：转化为生产资料的那部分资本，在生产过程中不改变价值量，只是把价值转移到新产品中去
  • 可变资本（v）：转化为劳动力的那部分资本，在生产过程中改变价值量，不仅能再生产出自身的等价，还能生产出剩余价值
商品价值 = c + v + m
（c = 不变资本，v = 可变资本，m = 剩余价值）

这一划分揭示了剩余价值的真正来源是可变资本，即工人的活劳动。它证明了剩余价值不是全部资本的产物，而仅仅是可变资本的产物。

剩余价值的生产方法

绝对剩余价值的生产

在必要劳动时间不变的条件下，通过绝对延长工作日长度（或提高劳动强度）来生产剩余价值的方法。工作日越长，剩余劳动时间就越多，剩余价值量就越大。

工作日的长度是有界限的：最低界限是必要劳动时间，最高界限取决于工人的生理界限和社会的道德界限。无产阶级为缩短工作日进行了长期的斗争。

相对剩余价值的生产

在工作日长度不变的条件下，通过缩短必要劳动时间、相应延长剩余劳动时间来生产剩余价值的方法。缩短必要劳动时间的条件是降低劳动力的价值，即降低维持工人生活所必需的生活资料的价值。

相对剩余价值的生产是通过提高社会劳动生产率来实现的。各个资本家追逐超额剩余价值（个别价值低于社会价值的差额）的竞争，最终导致社会劳动生产率的普遍提高，从而产生相对剩余价值。

相对剩余价值生产的三个阶段

  • 简单协作：许多工人在同一资本家指挥下协同劳动
  • 工场手工业：以分工为基础的协作，工人专门从事某种操作
  • 机器大工业：以机器体系为基础的协作，标志着资本主义生产方式的真正确立

第三章 工资

工资的实质

从现象上看，工资似乎是"劳动的价格"，是工人全部劳动的报酬。但实际上，工资是劳动力的价值或价格，而不是劳动的价值或价格。

劳动力和劳动是两个不同的概念。劳动力是人的劳动能力，是商品，有价值；而劳动是劳动力的使用，它本身不是商品，没有价值。

工资的本质：工资不是劳动的价值或价格，而是劳动力的价值或价格的转化形式。工资形式掩盖了工作日分为必要劳动时间和剩余劳动时间这一事实，造成了工人全部劳动都是有偿劳动的假象。

工资形式

  • 计时工资：按工人的劳动时间支付的工资，是劳动力日价值、周价值等的转化形式
  • 计件工资：按工人生产的产品数量或完成的工作量支付的工资，是计时工资的转化形式

计件工资更有利于资本家对工人的剥削，因为它为资本家提供了测量劳动强度的尺度，并便于实行层层剥削的包工制度。

相对工资和实际工资下降的趋势

相对工资是指工人所得工资在工人所创造的新价值中所占的比例。随着资本主义的发展，相对工资有下降的趋势。

实际工资是指工人用货币工资所能购买到的生活资料和服务的数量。从长期看，实际工资也有下降的趋势，这是因为通货膨胀、赋税加重等因素。

第四章 资本积累及其历史趋势

资本积累的本质

资本积累就是把剩余价值转化为资本，即剩余价值的资本化。资本积累是资本主义扩大再生产的唯一源泉。

通过对资本主义简单再生产的分析可以揭示出：资本家的全部资本都是工人创造的；工人的个人消费也是从属于资本的；资本主义再生产同时也就是资本主义生产关系的再生产。

资本的有机构成

资本的构成可以从两方面考察：

  • 价值构成：不变资本和可变资本的比例（c:v）
  • 技术构成：生产资料数量和使用这些生产资料所需的劳动力数量之间的比例

由资本技术构成决定并反映技术构成变化的资本价值构成，叫做资本的有机构成。随着资本主义的发展，资本有机构成有不断提高的趋势。

资本的积聚和集中

资本积聚：个别资本通过剩余价值资本化来增大自己的资本总额

资本集中：把若干已经存在的规模较小的资本合并成为大资本。竞争和信用是资本集中的两个最强有力的杠杆。

产业后备军

随着资本积累和资本有机构成的提高，资本对劳动力的需求相对减少，产生相对过剩人口（即产业后备军）。相对过剩人口有三种基本形式：

  • 流动形式：暂时失业的工人
  • 潜在形式：农业中的过剩人口
  • 停滞形式：就业极不稳定的劳动者

产业后备军不仅是资本积累的产物，也是资本主义生产方式存在和发展的必要条件。

无产阶级的贫困化

资本积累的一般规律是：一极是财富的积累，另一极是贫困的积累。无产阶级的贫困化表现为相对贫困化和绝对贫困化两个方面。

相对贫困化是指无产阶级在国民收入中所得份额的减少；绝对贫困化是指无产阶级生活状况的绝对恶化。

资本主义积累的历史趋势

资本积累过程一方面造成了社会化大生产，另一方面又使生产资料越来越集中到少数资本家手中。生产的社会化和资本主义私人占有之间的矛盾日益尖锐，最终必然导致资本主义制度的灭亡。

"生产资料的集中和劳动的社会化，达到了同它们的资本主义外壳不能相容的地步。这个外壳就要炸毁了。资本主义私有制的丧钟就要响了。剥夺者就要被剥夺了。" —— 马克思《资本论》第一卷

第五章 资本的循环和周转

产业资本循环的三个阶段

产业资本在运动中依次经过三个阶段，采取三种职能形式：

第一阶段：G — W   （购买阶段，货币资本转化为生产资本）
第二阶段：... P ...   （生产阶段，生产资本转化为商品资本）
第三阶段：W' — G'   （售卖阶段，商品资本转化为货币资本）

其中生产阶段是起决定作用的阶段，因为只有在生产过程中才能创造价值和剩余价值。

产业资本循环的三种形态

  • 货币资本的循环：G—W...P...W'—G'，表明资本运动的目的是为了价值增殖
  • 生产资本的循环：P...W'—G'—W...P，表明生产过程的连续性
  • 商品资本的循环：W'—G'—W...P...W'，表明全部商品的实现

产业资本的现实循环是这三种循环形式的统一。要使循环顺利进行，必须保持产业资本三种形式在空间上并存、在时间上继起。

流通时间和流通费用

流通时间是资本处于流通领域的时间，包括购买时间和售卖时间。流通时间不创造价值和剩余价值，它是对生产时间的一种限制。

流通费用包括：

  • 纯粹流通费用（买卖时间、簿记费用、货币费用）——不创造价值
  • 保管费用——在一定程度上加入商品价值
  • 运输费用——创造价值，属于生产性流通费用

资本的周转

周转时间和周转次数

资本周转是不断重复、不断更新的资本循环。资本周转时间等于生产时间和流通时间之和。

周转次数（n）= 一年（365天）/ 周转时间（天）

固定资本和流动资本

按价值的周转方式，生产资本可分为：

  • 固定资本：以厂房、机器、设备等劳动资料形式存在的生产资本，其价值逐渐转移、分次收回
  • 流动资本：以原料、燃料、辅助材料等劳动对象和劳动力形式存在的生产资本，其价值一次转移、一次收回

重要区分：固定资本和流动资本的划分与不变资本和可变资本的划分不同。前者是根据价值周转方式的不同，后者是根据在价值增殖过程中所起作用的不同。固定资本和流动资本中都包含不变资本的部分，只有流动资本中的劳动力部分才是可变资本。

垫付资本的总周转

垫付资本的总周转速度取决于各部分资本的周转速度及其在总资本中所占的比重。

垫付资本总周转次数 = (一年内固定资本周转总额 + 一年内流动资本周转总额) / 垫付资本总额

可变资本的周转对剩余价值的影响

可变资本的周转速度直接影响年剩余价值量和年剩余价值率。周转速度越快，同量可变资本在一年内剥削的劳动力就越多，年剩余价值量就越大。

年剩余价值量（M）= m' x v x n
年剩余价值率（M'）= M / V = m' x n
（m' = 剩余价值率，v = 可变资本，n = 周转次数）

第六章 社会资本的再生产和流通

个别资本和社会资本

个别资本是独立发挥职能的资本。社会资本是互相联系、互相依存的个别资本的总和。社会资本的运动不仅包括生产消费，还包括个人消费；不仅包括资本流通，还包括一般商品流通。

简单再生产条件下社会产品的实现

社会总产品按价值分为：c（不变资本）+ v（可变资本）+ m（剩余价值）；按实物形式分为：第一部类（生产资料）和第二部类（消费资料）。

简单再生产的基本实现条件：
I(v+m) = IIc
即：第一部类的可变资本加剩余价值等于第二部类的不变资本

由此还可引申出两个条件：

  • I(c+v+m) = Ic + IIc（第一部类全部产品价值等于两大部类不变资本之和）
  • II(c+v+m) = I(v+m) + II(v+m)（第二部类全部产品价值等于两大部类可变资本与剩余价值之和）

扩大再生产条件下社会产品的实现

扩大再生产的基本实现条件：
I(v+Δv+m/x) = II(c+Δc)
即：第一部类原有可变资本加追加可变资本加资本家个人消费等于第二部类原有不变资本加追加不变资本

扩大再生产的实现条件表明，社会生产两大部类之间必须保持一定的比例关系，否则社会产品的实现就会遇到困难。

资本主义条件下社会产品实现的矛盾

在资本主义制度下，由于生产的社会化和私人资本主义占有之间的矛盾，社会产品的实现经常遇到困难。这种矛盾表现为：生产无限扩大的趋势和劳动群众有支付能力的需求相对缩小之间的矛盾，最终必然导致经济危机。

第七章 平均利润和生产价格

生产成本和利润

资本主义企业生产的商品价值 = c + v + m。对资本家来说，c和v是他预付的资本，因此c+v构成商品的生产成本（k）。商品价值 = k + m。

当剩余价值不被看作是可变资本的产物，而被看作是全部预付资本的产物时，它就转化为利润（p）。利润是剩余价值的转化形式。

W = c + v + m = k + m = k + p
利润率 p' = m / (c + v)

利润率和影响利润率的因素

利润率是剩余价值与全部预付资本的比率。影响利润率的因素有：

  • 剩余价值率（同方向变化）
  • 资本有机构成（反方向变化）
  • 资本周转速度（同方向变化）
  • 不变资本的节省（同方向变化）

利润转化为平均利润

不同部门的利润率不同，资本家为了追逐更高的利润，会在不同部门之间展开竞争，资本从利润率低的部门转移到利润率高的部门。这种竞争导致各部门的利润率趋于平均化，形成平均利润率。

平均利润率 = 社会剩余价值总额 / 社会总资本
平均利润 = 预付资本 x 平均利润率

利润转化为平均利润，进一步掩盖了剩余价值的真正来源，造成利润是全部资本的产物的假象。

价值转化为生产价格

随着利润转化为平均利润，价值就转化为生产价格。生产价格 = 生产成本 + 平均利润。

生产价格形成后，商品不再按价值出售，而是按生产价格出售。但从全社会看，生产价格总额等于价值总额，平均利润总额等于剩余价值总额。生产价格是价值的转化形式。

平均利润率下降趋势的规律

随着资本主义的发展，资本有机构成不断提高，导致平均利润率有下降的趋势。但这一规律并不排除利润量的绝对增加，因为总资本的增长速度可能超过利润率下降的速度。

阻碍利润率下降的因素包括：劳动生产率的提高使工人必需品便宜、相对过剩人口的存在、对外贸易的发展、股份资本的增加等。

第八章 商业资本和商业利润

商业资本是产业资本的独立部分

商业资本是专门从事商品买卖、以获取利润为目的的资本。它是产业资本中商品资本的独立化形式。

商业资本的职能是执行商品资本的职能——实现商品的价值和剩余价值。

商业利润和商业价格

商业利润是商业资本家从产业资本家那里以低于商品价值（生产价格）的价格购买商品，再以商品价值（生产价格）出售所获得的差额。

从表面上看，商业利润似乎是在流通中产生的。但实际上，商业利润的来源是产业工人在生产中创造的剩余价值。产业资本家以低于生产价格的价格把商品卖给商业资本家，商业资本家再按生产价格出售，其中的差额就是商业利润。

商业利润 = 商业价格 - 购买价格
商业利润率 = 平均利润率（通过部门间竞争趋于平均化）

商业的纯粹流通费用及其补偿

纯粹流通费用是因商品买卖本身而支出的费用，如广告费、簿记费、商业职工工资等。这些费用不创造价值，但需要从剩余价值中得到补偿。

纯粹流通费用从产业工人创造的剩余价值中扣除来补偿，这降低了对社会总剩余价值的分配比例。

资本主义商业的形式

  • 批发商业：在企业之间进行的商品大宗买卖
  • 零售商业：直接把商品卖给消费者的商业
  • 对外贸易：超越国界的商品买卖活动
