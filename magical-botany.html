Magical Botany
"存在于每一片叶子、每一朵花、每一棵草木之中的古老魔法与智慧" — 基于 Gerina Dunwich《Herbal Magick》

植物占卜学 / Botanomancy

植物占卜（Botanomancy）源自希腊语"botane"（草药），是人类最古老的预言方法之一。自古以来，女巫、德鲁伊和占卜师通过观察植物的形态、燃烧方式和烟迹来解读未来的讯息。

叶脉占卜 / Phyllomancy

通过解读树叶上的脉络图案来洞察未来事件或揭示未知之事。占卜者仔细观察叶脉的分支走向、交叉点和纹路的深浅，从中寻找神谕的暗示。

燃烧占卜 / Causimomancy

通过燃烧植物产生的灰烬来解读预兆。植物在火中燃烧的速度——若缓慢冒烟或无法燃烧被视为凶兆，若迅速燃烧则为吉兆。该占卜法有多种变体：

Capnomancy（烟占术）：解读燃烧植物产生的烟雾形态。起源于古巴比伦，祭司将雪松枝放在热炭上或投入火中，仔细解读烟雾的象征意义。烟雾直直升天被视为吉兆；烟雾低垂则为凶兆；若烟雾触及地面，则是神灵或诸神发出的警告。

Daphnomancy（月桂占卜）：通过燃烧月桂木或月桂叶产生的烟雾和声音来解读预兆。该占卜法源于前基督教时代的罗马，与罗马皇帝种植的月桂圣林有关。

Libanomancy（焚香占卜）：解读焚香烟雾的占卜法，可追溯至古巴比伦的魔法师。烟雾向右飘散预示你将在对抗中获胜；向左飘散则表示对手将获胜。烟雾聚集是成功和财运的吉兆；若烟雾分散，则预示财务损失。

玫瑰花瓣占卜 / Phyllorhodomancy

古希腊流行的占卜法，通过观察玫瑰花瓣来预测未来。占卜者选择一片凹形玫瑰花瓣，提出一个是否问题，然后将花瓣放在右手掌中，用力合掌拍击一次。若花瓣爆裂，答案为肯定；若未爆裂，则为否定回答。

樱桃树占卜 / Cherry Tree Divination

"樱桃树，我摇动你，
樱桃树，请告诉我，
我将活到多少岁？
落果之数即答案。"

在仲夏夜，围绕结满果实的樱桃树顺时针跑三圈，然后用力摇晃树干。数一数掉落在地上的樱桃数量——每一颗代表你将活到的一岁年龄。

橡子占卜 / Acorn Divination

在满月之夜，取两粒橡子，一粒刻上你的姓名首字母，另一粒刻上未婚夫/妻的首字母。将两粒橡子放入盛满水的坩埚中，保持3英寸距离，然后观察它们的移动：若它们相互靠近，预示婚礼即将到来；若相互远离，表示将分手；若保持不动，需改日再占。

万圣节植物占卜 / Halloween Plant Divinations

鼠尾草婚姻占卜：在万圣节午夜时分，独自走进花园，一言不发地摘下12片鼠尾草叶——每敲响一次钟声摘一片。当第12次钟声响起时摘下第12片叶子，你未来丈夫的面容将在你眼前显现。

燕麦子女占卜：在万圣节午夜，独自走进燕麦田。闭眼顺时针自转三圈，伸手随机拔三株燕麦。数第三株上的谷粒数——这将是你会拥有的子女数量。

落叶许愿：在万圣节接住一片飘落的秋叶，许下愿望——传说这个愿望一定会实现。

更多植物占卜法

月桂叶恋人占卜：在圣约翰前夜，取一片月桂叶，用针刻上爱人的名字。将叶子别在睡衣上贴近心脏的位置入睡。早晨检查叶子：若变为棕色，表示爱人一直忠实于你；若颜色未变，则表示爱人已经或将要欺骗你。

苹果皮占卜：将苹果削成一整条不断的皮。在粉色魔法蜡烛的光芒下，右手拿果皮，念咒三遍后从右肩抛向身后。若果皮落地形成字母形状，则是未来配偶姓氏的首字母。若果皮断裂，预示你将永不结婚。

三叶草占卜：摘一片两叶三叶草放入右鞋中。女性遇到的第一个年轻男性将拥有与你未来丈夫相同的名字或首字母。

蓍草爱情占卜：在五朔节前夜或新月之夜摘10根蓍草茎。睡前将9根放在枕头下，第10根从左肩抛向身后，同时念诵咒语，将梦见未来真爱。

冬青梦占卜：在周五午夜，一言不发地走进花园，从雌性（光滑边缘）冬青植株上摘九片叶子。将每片叶子打结系在三角手帕中，放在枕头下入睡，将梦见未来配偶。

玫瑰梦占卜：在仲夏夜午夜钟声敲响12下时，一言不发地倒退走进花园，摘下最红的那朵盛开的玫瑰。用白纸包好藏在秘密处。在冬至日出时取出放在胸前——据说你将来的丈夫会来把它取走。

爱情魔法 — 幸运之手招爱咒

材料：幸运手根（兰花根）、玫瑰油、带盖小罐
在周五（金星之日），月亮处于盈相且在金星时刻，将幸运手根放入装满玫瑰油的小罐中。盖上盖子，念诵咒语三次："盈月金星时，注此根以神秘之力。令其无阻运作，吸引真爱降临。"让根在玫瑰油中浸泡七天七夜。接下来的周五，在金星时刻取出根，每日每夜贴身佩戴。
最佳时间：周五、盈月、金星时刻

爱情魔法 — 长生草招爱咒

材料：新鲜长生草（Houseleek）、玫瑰油或爱情魔法油、魔法袋
在魔法袋中装满新鲜长生草，同时心中满怀对心上人的浪漫思念。用三滴玫瑰油或爱情魔法油涂抹袋子，然后每日每夜贴身佩戴。每三天更换新鲜草药并重新涂油。
最佳时间：新月开始效果最佳

爱情魔法 — 吉普赛洋葱爱情咒

材料：一个洋葱或郁金香球茎、干净的新花盆、新土
将洋葱或郁金香球茎种植在干净的未使用过的花盆中，同时念诵心上人的名字。每天日出和日落时分对着花盆念诵："如此根生长，如此花绽放，愿他/她的心，转向我！"随着时间的推移，你心爱之人将越来越倾向于你。
最佳时间：日出和日落时分

爱情魔法 — 莳萝忠诚占卜

材料：圣约翰草、门框和窗框
在圣约翰日（6月24日）采摘圣约翰草，将其悬挂在房屋和谷仓的门窗上方，可防止邪恶精灵进入。随身携带或在魔法袋中携带圣约翰草，可抵御最邪恶的幽灵和恶魔实体。
最佳时间：圣约翰日（6月24日）

招财魔法 — 海角约翰招财符

材料：海角约翰根（High John the Conqueror）、2美元闰年纸币、银色水星10美分硬币、绿色法兰绒魔法袋、海角约翰油
将2美元闰年纸币包裹海角约翰根和银色水星10分硬币（注意朝自己方向折叠，不可朝外）。放入绿色法兰绒魔法袋中，每日滴三滴海角约翰油或招财魔法油。无论睡眠时都必须随身携带。
最佳时间：周三水星时刻制作

招财魔法 — 中国招财咒

材料：三枚硬币、红纸、一盆圆叶植物
将三枚硬币用红纸包裹，埋在圆叶盆栽的土壤中。植物越大，咒语效力越强。务必保持植物健康快乐，及时移除枯萎或死亡叶片，否则咒语的正面效果可能逆转。
最佳时间：新月之时

招财魔法 — 啤酒花好运咒

材料：干啤酒花
用干啤酒花编织花环，悬挂在壁炉架或炉膛上方，可吸引好运并驱散厄运。每年啤酒花采摘季节更换新花环，防止好运耗尽。此外，将干啤酒花塞入枕套可治疗失眠和驱散噩梦。
每年采摘季节更换

幸运植物

橡子 / Acorn

幸运
将橡子放在窗台上可守护房屋免遭火灾和雷击。这一迷信可追溯至北欧神话——雷神索尔曾在橡树下躲避雷雨。
将橡子放在窗台即可。

苹果树 / Apple Tree

幸运
"一日一苹果，医生远离我。"在第十二夜向苹果树祝酒（wassailing）可驱散邪灵。在挪威异教徒中，吃苹果可获得"永生智慧"。

罗勒 / Basil

幸运
罗勒被称为"女巫之草"，在印度文化中与图尔西女神相关，被视为神圣的植物，能带来好运、爱情和保护。

月桂 / Bay Laurel

幸运
月桂叶放在枕头下可带来预知之梦。二月十三日夜，将五片月桂叶放在枕头下，未婚女子将梦见未来丈夫。

桦树 / Birch

幸运
桦树象征女神或阴性原则，是五朔节圣火九种圣木之一，象征新生与净化。

三叶草 / Clover

极其幸运
四叶三叶草是极其幸运的象征。两叶三叶草可用于爱情占卜。

接骨木 / Elder

幸运（但需谨慎）
接骨木是德鲁伊用于祝福和诅咒的树木。在门窗上悬挂接骨木可抵御恶魔，但燃烧接骨木则会招引恶魔入门。

蕨类 / Fern

幸运
蕨类植物具有吸引好运的力量。男子用雄性蕨籽酿造爱情药水给心仪女子，女子用雌性蕨籽酿造给心仪男子。

大蒜 / Garlic

保护与幸运
大蒜驱赶吸血鬼和邪灵的力量举世闻名。据说五月采摘的大蒜最有效。大蒜还能吸收疾病和负面能量。
随身携带大蒜或挂在门口。

榛树 / Hazel

幸运
榛树象征智慧，是五朔节九圣木之一。凯尔特树月中有以榛树命名的月份。

石楠 / Heather

冬青 / Holly

长生草 / Houseleek

种植在房屋外墙或屋顶上。

常春藤 / Ivy

幸运（室外）/ 不吉（室内）
常春藤象征婚姻和忠贞。但在维多利亚花语中，常春藤代表婚姻和忠诚。将常春藤带入室内会带来厄运。

茉莉 / Jasmine

在圣约翰日采摘，挂在门窗上。

幸运

魔法师之草
马鞭草被称为"魔法师之草"，在仲夏或天狼星升起时采摘最为有效。用于爱情、保护和财运魔法。

紫罗兰 / Violet

幸运
紫罗兰用于爱情魔法、保护和吸引好运。在金星统治下，与爱情和美丽相关。

柳树 / Willow

保护
柳树长期以来被珍视为抵御巫术和邪眼的天生保护者。触摸柳树可确保好运。但不可在柳树下泄露秘密，否则秘密会被柳树"泄露"出去。
触摸柳树带来好运；柳木不可在篝火之夜焚烧。

蓍草 / Yarrow

保护与占卜
将蓍草撒在门槛上，魔鬼不敢进入你的房屋。蓍草用于驱散邪灵和负面能量，以及占卜爱情。
撒在门槛上可辟邪；放在枕头下可梦见未来爱人。

不吉植物

颠茄 / Belladonna (Deadly Nightshade)

剧毒
颠茄被称为"恶魔之樱桃"和"女巫之莓"，在土星统治下，与黑卡蒂女神相关。具有剧毒，在魔法中常用于飞行药膏。
避免接触和误食；不可带入家中。

黑莓（10月11日后） / Blackberry (after Oct 11)

魔鬼的诅咒
10月11日（旧历米迦勒节）后采摘或食用黑莓会遭厄运。传说恶魔在很久以前的这一天跌入黑莓丛，对黑莓植物施下了诅咒。
10月11日后不可采摘或食用黑莓。

金雀花 / Broom

五月最凶
金雀花长期被视为不祥之兆，五月尤其不吉。用开花金雀花扫屋子（或带入室内）会"把头扫走"。
五月不可将金雀花带入室内。

仙人掌（作为礼物） / Cactus (as gift)

带刺的礼物
在许多文化中，带刺的仙人掌作为礼物被认为会刺破友谊或爱情，带来厄运。
避免将仙人掌作为礼物赠送。

水仙（单株） / Daffodil (single)

不吉之兆
若春天或夏天看到的第一朵水仙花向你的方向低头，这预示着接下来一年的厄运。此外，带进家中的报春花或水仙花总数不可少于13朵。
避免单株水仙；水仙总数不少于13。

毛地黄 / Foxglove

精灵的指套
毛地黄被称为"精灵指套"、"精灵围裙"，在金星和水元素统治下。虽美丽但全株有毒，与仙灵世界相关。
不可误食；尊重其毒性。

毒参 / Hemlock

巫师之草
毒参被称为"巫师之草"，在土星和水元素统治下，献给黑卡蒂女神。苏格拉底即死于毒参。剧毒。
剧毒植物，避免接触。

天仙子 / Henbane

恶魔之眼
天仙子被称为"恶魔之眼"，在土星统治下。中世纪女巫的飞行药膏成分之一，全株有毒。
剧毒植物，避免接触。

绣球花 / Hydrangea

老处女之咒
据英国民间传说，绣球花对想找丈夫的年轻女士是不吉利的植物。让绣球花在房屋附近（尤其是前门旁）生长，会诅咒女儿孤独终生。
避免在房屋附近种植绣球花。

常春藤（室内） / Ivy (indoors)

室内不祥
将常春藤带入室内会带来厄运。采摘教堂墙上的常春藤叶会招致疾病。房屋墙上常春藤无故枯萎预示家中将发生死亡。
不可将常春藤带入室内。

曼德拉草 / Mandrake

尖叫的根
传说曼德拉草被拔出地面时会发出刺耳的尖叫，听到的人会发疯或痛苦死亡。中世纪魔法师用狗来拔曼德拉草作为安全措施。曼德拉草根形似人体，用于爱情和生育魔法。
采摘时需谨慎；传统上用狗来拔取。

夹竹桃 / Oleander

剧毒
夹竹桃全株剧毒，在土星和土元素统治下。不可食用或接触其汁液。
剧毒植物，远离儿童和宠物。

一品红 / Poinsettia

室内不吉
一品红的乳汁有毒，在一些传统中被视为不吉利的室内植物。
避免接触其乳汁；如有宠物需特别注意。
